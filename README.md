# Discount-Calculator
A simple GUI-based discount calculator built in Python and Tkinter that calculates the price and quantity of an item after applying a discount percentage. The results are displayed in Indonesian Rupiah (Rp).

# Screenshot preview
![Hasil Kalkulator Diskon](https://github.com/user-attachments/assets/980b9748-76fb-4d23-97cd-682260b47005)

# Features
⦁	Easy-to-use graphical interface
⦁	Calculate prices, quantities, and discounts using percentage input
⦁	Display results in Rupiah currency format with thousands separator
⦁	Input validation with error handling
⦁	Clean and modern design

# Requirements
⦁	Python 3.12
⦁	Tkinter (included in Python standard library)

# Installation
1.	Clone or download this repository
2.	Python's standard library (tkinter) for GUI

# Usage
Run the application:
python "Discount Calculator.py"

# How to Use
1.	Enter the original price in the "Original Price" field.
2.	Enter the number of items in the "Quantity of Items" field.
3.	Enter the discount percentage in the "Discount (%)" field.
4.	Click the "Calculate" button.
5.	The final price will be displayed in Rupiah.

# Example
⦁	Original Price: 100000
⦁	The amount of goods: 10
⦁	Discount (%): 15
⦁	Final Price: Rp 850,000.00
